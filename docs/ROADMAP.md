# Roadmap

Living map for **Multiwindow Autobrowser**. Not a dated promise schedule.

- **Distribution:** open source on GitHub — no Play Store
- **Hardware:** flagship phones, foldables, and tablets (multi-WebView + optional Gemma 4)
- **Canonical copy:** [README § Roadmap](https://github.com/Pixelgamer4k/autobrowse-android#-roadmap)

| Horizon | Meaning |
|---------|---------|
| **Near** | Next major releases — polish what exists, ship high-impact UX |
| **Mid** | New subsystems — multi-agent, canvas intelligence, creative outputs |
| **Far** | Platform bets — APIs, community scale, hardware-specific depth |
| **Moonshot** | Speculative — worth designing toward, not committing dates |
